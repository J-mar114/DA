# Tesda Batch 3 | Jbudduan

Developer: Jaymar Budduan

A polished digital gallery website for showcasing artwork, certificates, and visual assets in a modern and elegant presentation.

## Features
- Interactive gallery cards
- Lightbox image viewer
- Download button for full-resolution images
- Light/Dark theme toggle
- Smooth scroll reveal animations
- Scroll progress indicator
- Custom certificate section with artistic styling

## Project Structure
- index.html — main webpage structure
- styles.css — custom styles and visual effects
- script.js — interactive behavior such as theme switching, lightbox, and animations
- dist/output.css — generated Tailwind CSS file
- src/input.css — Tailwind source entry file
- package.json — local Tailwind build scripts

## How to Run
Open index.html in a browser.

If you want to rebuild the Tailwind CSS file:

```bash
npm install
npm run build:css
```

## Notes
- The site uses local Tailwind CSS instead of the CDN script for a production-friendly setup.
- Images used in the gallery are loaded from the project folder.

## Preview
You can open the website locally by launching index.html in your browser.
